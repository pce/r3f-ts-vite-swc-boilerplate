
# R3F Boilerplate


Example:

    gcl --depth 1 git@github.com:pce/r3f-ts-vite-swc-boilerplate.git
    yarn 
    yarn dev 
    o

    # in another terminal tab    
    code .

Build    

    yarn build 


Options: adjust basePath at index.html <base href="./"> (ie over env variable) and vite configs base, default: base: './'
